# EasyClick  Html  与 EasyClick  Js 交互



## 介绍

> Html 与 Ec js 交互
>
> 仓库中例子使用了vue . 
>
> 可以把vue 换成 Jquery ,或者是其他的框架

## 注意事项

> 必须使用 EC 5.0.0  以上的版本
>
> ! ! ! ! ! 低版本 有bug  不能使用 JavaAdapter

## jar包问题

> 仓库上传代码的时候 
>
> 由于 android.jar  和 opencv-4.3.0.jar  太大.
>
> 而且对于本项目没有什么帮助, 
>
> 所以 移除了 这两个 jar
>
> 如果要用于生产环境, 
>
> 建议 自己新创建一个项目,
>
> 将 核心代码 复制到自己的项目中使用
